# TechNova Solutions Website

A simple responsive company website built with HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Home Page
- About Page
- Contact Form
- Ready for Docker Deployment
- Ready for Jenkins CI/CD Pipeline

## Technologies

- HTML5
- CSS3
- JavaScript

## Run Locally

Simply open `index.html` in your browser.

Later, this project will be containerized using Docker and automatically deployed to AWS EC2 through Jenkins.
